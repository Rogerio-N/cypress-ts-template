![TypeScript](https://img.shields.io/badge/Typescript-blue?style=for-the-badge&logo=typescript&logoColor=white)

This is a template repository. Its purpose is to be used in new projects that uses Cypress with TypeScript.  

### Pre-requisites

- [Node 16](https://nodejs.org/en/download/)

### Setup

1 - First you need to install the [Yarn package manager](https://classic.yarnpkg.com/lang/en/docs/install/#windows-stable).
To install, open your terminal and run the command: `npm install --global yarn`.

> If you have problems during this install, open just click on the yarn package manager and you will see other installation methods

2 - Open the project and install its dependencies, to do this run the command in the terminal: `yarn`

3 - Once the installation finishes, you can open the cypress window. Run the command: `yarn cypress open`
